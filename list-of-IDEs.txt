1.Eclipse- Eclipse is  fully customizable and extensible,it is also free. Great for hobbyists and casual programmers wanting to just hack out some code in their free time.A downside to eclipse is that it has a steep learning curve, because there are alot of upgrades and addons for it, which can be overwhelming for some.
2.NetBeans-This program is extensible and free, and it compiles code which is a bonus. When installing Netbeans it comes with a Java server called GlassFish that you could deploy on your computer for your testing environment. It has great extensibility, which is a must have for the development environments' survival.The downside to this IDE is that it can slow down your computer. It is also a tough software to master.
3.gedit- This is the official editor of the gnome desktop and comes installed once the system is setup. Very useful for the user that is casually editing code and not undertaking major projects.The major con of this IDE is that it doesn't compile code.
4.Rapid SQL- Rapid SQL is the intelligent SQL IDE empowering database developers and DBAs the ability to create high-performing SQL code on all major databases from a single interface. This toolset simplifies SQL scripting, query building, object management, debugging and version control with intuitive, innovative tools. 
5.Vim-Vim is a popular open-source, cross-platform, and free editor, with good support for editing Perl code.By default Vim behaves very much like the ubiquitous UNIX editor vi, which may seem foreign to people used to the conventions of modern desktop environments.
